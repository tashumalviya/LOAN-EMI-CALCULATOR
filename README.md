# LOAN-EMI-CALCULATOR
Loan Emi Calculator built using HTML, CSS & JAVASCRIPT

## Project Description
A responsive Loan EMI Calculator web application built using HTML, CSS, and JavaScript that calculates monthly EMI and displays a detailed amortization schedule including interest and principal breakdown.

## Features
- Calculate monthly EMI instantly
- Displays full amortization schedule (month-wise)
- Interest and principal breakdown
- Multiple loan types (Car, Home, Bike, Personal)
- Input validation for loan amount and tenure
- Simple and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## EMI Calculation Formula
The calculator uses the standard EMI (Equated Monthly Installment) formula used in banking and financial systems:

EMI = P × r × (1 + r)^n / ((1 + r)^n − 1)

Where:
- P = Loan Amount (Principal)
- r = Monthly Interest Rate (Annual Rate / 12 / 100)
- n = Loan Tenure in Months

## How It Works
The user enters the loan amount, interest rate, and tenure.  
The system calculates:
- Monthly EMI
- Total Interest Payable
- Total Payment Amount
- Month-wise amortization schedule

## Use Case
This project helps users estimate monthly loan payments and understand interest distribution over the loan tenure, similar to real banking EMI calculators.
